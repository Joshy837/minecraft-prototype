# Feature Ideas

## Gameplay
- [ ] Crafting system — crafting table block + 3×3 grid UI
- [ ] Hunger / food system — food items, hunger bar alongside health
- [ ] Mobs — passive (cows, chickens) or hostile (zombies at night)
- [ ] Tool tiers — wooden / stone / iron with hardness multipliers
- [ ] Armor — equipment slots, damage reduction
- [ ] Chest block — persistent storage, opens inventory overlay
- [ ] Enchanting table
- [ ] Experience / XP orbs

## World Generation
- [ ] Caves — 3D noise carving below terrain
- [ ] Ore generation — coal, iron, gold at depth bands
- [ ] More biomes — desert, plains, ocean
- [ ] Dungeon / structure generation
- [ ] Weather — rain / snow particles
- [ ] Lava block + flow simulation (mirrors water BFS)

## Building / Blocks
- [ ] Stairs and slabs — new block shapes
- [ ] Ladders — climbable surface blocks
- [ ] Trapdoors
- [ ] Glass pane / iron bars — thin geometry
- [ ] Fences + fence gates
- [ ] Signs — text rendering on a block face
- [ ] Beds — set spawn point

## UI / Polish
- [ ] Block placing preview — ghost mesh at placePos
- [ ] Mini-map — canvas overlay of nearby chunks
- [ ] Death screen + respawn flow
- [ ] Settings / keybindings menu
- [ ] F3 debug overlay (coords, FPS, chunk info)
- [ ] Hotbar item count badge
